王小波的情话

发表于 2020-04-15 阅读次数： Valine：
本文字数： 719 阅读时长 ≈ 1 分钟

王小波对文坛的影响之大，甚至有人称：如果王小波还在世，他将是比莫言更早获得诺贝尔文学奖的中国作家。

而他更为人知的，是天赋异禀的超能情话。毫不夸张的说，王小波本人就是一部情话百科全书。他和妻子李银河的故事，从他写给李银河的情话就可以窥见。

/

你好哇！李银河！

做梦也想不到我把信写到五线谱上吧？

五线谱是偶然来的，你也是偶然来的。

不过我给你的信值得写在五线谱里呢。

但愿我和你，是一支唱不完的歌。

/

告诉你，一想到你，

我这张丑脸上就泛起微笑。

还有在我安静的时候，

你就从我内心深处浮现，

就好像阿芙罗蒂从浪花里浮现一样。

/

我和你就像两个小孩子，

围着一个神秘的果酱罐，

一点一点地尝它，看看里面有多少甜。

/

我的勇气和你的勇气加起来，

对付这个世界足够了吧！

/

我现在不坏了，我有了良心。

我的良心就是你。

/

我希望你爱我的全部肉体，

我愿意它因为你而变得美。

/

好了，不跟你说话了，我记仇了。

我星期一给你写的信你都不回。

我记得我在信尾说：

我们要占有世界上最美好的东西，

可是你不理我，你不愿意听我说话。

/

我把我整个灵魂都给你，

连同它的怪癖，耍小脾气，

忽明忽暗，一千八百种坏毛病。

它真讨厌，只有一点好，爱你。

/

你要是愿意，我就永远爱你。

你要是不愿意，我就永远相思。

/

不管我本人多么平庸，

我总觉得对你的爱很美。

/

海誓山盟是把现在的东西固定住，

两个人都成了活化石。

我们用不着它。我们要爱情长久。

真的，它要长久我们就老在一块，不分开。

/

要无忧无虑地去抒情，去歌舞狂欢，

去向世界发出我们的声音，

我一个人是不敢的，

我怕人家说我疯，有了你我就敢。

只要有你一个，就不孤独！

/

我们创造了记录——一星期不见的记录。

你感觉怎样？受得了吗？

连我都快受不了了。

让不断的思念把我们的火持续地烧下去吧。

加个鸡腿～
